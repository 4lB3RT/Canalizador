# Template: Post de planificación

## Estructura del post

```
[FECHA] 📋

[HOOK - 1-2 frases que plantean el problema o la oportunidad de forma directa y cercana]

[SOLUCIÓN - Qué vas a construir para resolverlo, en primera persona, storytelling]

[CTA - Frase que invite a seguir el progreso o a la acción]
```

## Ejemplo

```
2026-01-21 📋

Eres creador de contenido gaming y necesitas fotos profesionales para tus redes. Pero no tienes un setup real ni acceso a un fotógrafo.

Voy a construir un sistema donde subes tu foto y te devuelve 3 imágenes tuyas en un setup gaming: frontal, 30° izquierda y 30° derecha. Realistas y listas para usar.

Pronto en Canalizador.
```
